About FidgetRunner 2

Fidget Runner 2 is a sequel to the hit mobile game Fidget Runner. My brother made the original for a school project about 5 years ago so when I got the assignment to make a mobile game I immediately knew what I wanted to make.

Features

Motion Controls

Since this is a mobile game I wanted to implement something that only the phone can do. So inspired by the game "Send Me To Heaven" I decided to add a feature where you need to throw your phone in the air to jump. I also made a basic accelerating and decelerating depending on how you tilt your phone mechanic.

Skins

This is around the same time I started learning shaders, so I wanted to add different skins that you can get with the in game currency "Fidgets".

Obstacles

The core gameplay loop is simple: begin a new run, avoid obstacles, collect Fidgets, and use them to get new skins. The obstacles are procedurally generated so the course is different for every run you do.

Death Animation

Another fun thing I did is use the player for the backboard of the death screen. The play testers also liked this feature as the animation is quite humorous and dampens the frustration of dying.
