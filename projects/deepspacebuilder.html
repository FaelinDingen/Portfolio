About Deep-Space Builder

For my final exam at collage I needed to create a game about flying spaceships in a team of 4.
We created a ship building game where you go out to collect resources and use that to build new stuff on your ship.
I was in charge of the ship movement, menu screens, thruster and turret logic and a couple of other things.
There were quite a lot of challenges along the way but eventually it all worked out.
We got full marks on the project and it was a wonderfully final collage project.

Features

Interactive main menu

For the main menu I wanted a nice showcase of some cool things in the game.
I first added the hangar in the background then added some other atmospheric things like asteroid and fog.
I then added a couple of different enemy ships that attack the players mouse.
I positioned these so that they will be able to hit each other for some cool interactivity.

Ship builder

The ship builder is a big part of the game.
It has a lot of features like mirroring, snapping and selling components.
There are a lot of components that you could get.
They are split up into 4 classes: Fuel-Tanks, Thrusters, Turrets and Blocks.

Mining asteroids

To get resources you will need to go out and mine asteroids.
You can do this by shooting them and picking up the bits.

Late-game

Late-game you will most likely have a bit ship with a lot of turrets and thrusters.
Enemy ships will be attacking you and destroying components.
You need to survive as long as possible and see which rank you can get.
